## Stack

- JS Metaframework: Next.js 12;
- JS Runtime: Node 16;
- Checkout: Snipcart;
- CMS: Hygraph;
- GraphQL Client: Apollo Client;
- CSS Preprocessors: Saas;
- Media optimization: Cloudinary.
